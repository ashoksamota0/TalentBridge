# TalentBridge – Full Stack Job Portal

TalentBridge is a full-stack recruitment platform designed to connect job seekers and recruiters through a streamlined hiring experience. The platform enables candidates to discover opportunities, apply for jobs, and track their applications, while recruiters can manage job postings and review applicants from a centralized dashboard.

## 🚀 Live Demo

https://talent-bridge-portal.vercel.app

## 📂 GitHub Repository

https://github.com/ashoksamota0/TalentBridge

## ✨ Features

### Candidate Features
- Secure user authentication
- Browse and search job listings
- Apply for jobs online
- Upload resumes and supporting documents
- Track application status

### Recruiter Features
- Dedicated recruiter dashboard
- Create, edit, and manage job postings
- View candidate applications
- Review applicant profiles
- Manage hiring workflows efficiently

## 🔧 Installation

### Clone Repository

```bash
git clone https://github.com/ashoksamota0/TalentBridge.git
```

### Install Dependencies

Frontend:

```bash
cd client
npm install
```

Backend:

```bash
cd server
npm install
```

### Environment Variables

Create `.env` files for both client and server and configure the required API keys and database credentials.

### Run the Application

Backend:

```bash
npm start
```

Frontend:

```bash
npm run dev
```
